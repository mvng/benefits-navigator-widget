# Benefits Navigator Widget

San Diego County Benefits Navigator — 73 programs single-page static app.

## Deploy

Hosted via GitHub Pages at: https://mvng.github.io/benefits-navigator-widget/

## Squarespace Integration

Embed on any Squarespace page using a **Code Block**:

```html
<iframe 
  src="https://mvng.github.io/benefits-navigator-widget/" 
  width="100%" 
  height="900px" 
  style="border:none; border-radius: 12px;"
  title="Benefits Navigator"
></iframe>
```

Or use a **Script Injection** in Settings > Advanced > Code Injection > Page Header to load it directly.

## Updating Content

1. Edit `index.html` locally
2. `git add . && git commit -m "update" && git push`
3. GitHub Pages auto-deploys in ~60 seconds
